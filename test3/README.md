- elementy STL (Create_Character.cpp)
- zakresowa pętla for (Create_Character.cp)
- własny szablon klasy (lub funkcji) (Controller.cpp/.h)
- plik .h z deklaracja i .cpp z definicją metod
- dziedziczenie             |\
